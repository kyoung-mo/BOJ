## 폴더 구조
알고리즘 유형별로 분류하며, 파일명은 `문제번호_제목.c` 형식을 따릅니다.
```
BOJ/
├── Math/
├── Implementation/
├── BruteForce/
├── Sort/
├── BinarySearch/
├── Stack/
├── Queue/
├── DFS/
├── BFS/
├── DP/
├── Greedy/
└── ...
```
